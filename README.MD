# Spark

Dream game!

## Clone 

```
git clone --recurse-submodules https://github.com/IltwMyLife/Spark.git
```

## Setup

  * Download [CMake](https://cmake.org/download/)

## Build

```
mkdir build && cd build && cmake .. && cmake --build .
```

## Dependencies

  * [SFML](https://github.com/SFML/SFML.git)
